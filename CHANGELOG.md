# Changelog

## [1.0.0] - 2026-01-28
### Added
- Complete Python 3 migration and modernization
- Modular package structure (models, config, utils, attacks, core)

- Unit and integration tests (pytest)
- Strict type hints throughout codebase
- Robust error handling and logging
- Autoformatting (black) and linting (flake8)
- Packaging and distribution files (setup.py, pyproject.toml, MANIFEST.in, LICENSE)
- Documentation and README overhaul
